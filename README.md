# Hydrogen formation quench

Static GitHub Pages simulator for classical real-time evolution in bosonized 1+1D two-flavour QED.

This version is derived from `AlienTu/Classical-2-flavour-QED2-quench` but is kept in a separate repository. It adds selectable periodic, Dirichlet, and Neumann boundary conditions; a non-periodic spin-sector charge input; and zero-momentum nontrivial initial field profiles.
